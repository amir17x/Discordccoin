## 1️⃣ 💰 بخش اقتصادی (Economy)
### 🎯 هدف
ایجاد یه سیستم اقتصادی پویا برای مدیریت منابع مالی کاربران (Ccoin و کریستال) و تشویق به فعالیت اقتصادی.

### 📜 Embed بخش اقتصادی
- **عنوان**:  
  `💰 بخش اقتصاد`
- **توضیحات**:  
  `منابع مالی خودت رو مدیریت کن، سکه‌ها رو انتقال بده و با سرمایه‌گذاری سود به دست بیار! 🚀`
- **رنگ Embed**: سبز (#00FF00) برای حس ثروت.

#### 1. 💳 کیف پول (Wallet)
- **هدف**: ذخیره Ccoin کاربران برای استفاده روزمره.  
- **مکانیزم‌ها**:  
  - نمایش موجودی کیف پول.  
  - مالیات روزانه: 1% از موجودی کیف پول (برای تعادل اقتصادی).  
  - امکان دزدی: هر 4 ساعت، شانس 50% برای دزدیده شدن حداکثر 100 Ccoin (بخش دزدی توضیح داده می‌شه).  
- **پیام نمونه**:  
  ```
  💳 کیف پول تو:
  💰 موجودی: 500 Ccoin
  ⚠️ مالیات روزانه: 1% (5 Ccoin کسر شد)
  ```
- **محدودیت‌ها**:  
  - حداکثر موجودی کیف پول: 50,000 Ccoin (برای جلوگیری از احتکار).  
  - اگر موجودی بیشتر بشه، به بانک منتقل می‌شه (کارمزد 2%).  
- **نکات توسعه**:  
  - یه فیلد تو دیتابیس برای کیف پول بساز:  
    ```javascript
    userSchema.add({ wallet: { type: Number, default: 0 } });
    ```
  - برای مالیات روزانه، یه تایمر 24 ساعته پیاده‌سازی کن:  
    ```javascript
    schedule.scheduleJob('0 0 * * *', () => {
        users.updateMany({}, { $inc: { wallet: -wallet * 0.01 } });
    });
    ```

#### 2. 🏦 بانک (Bank)
- **هدف**: سپرده‌گذاری Ccoin برای کسب سود.  
- **مکانیزم‌ها**:  
  - سود ماهانه: 2% از موجودی بانک (هر 30 روز).  
  - کارمزد واریز: 1% (مثلاً واریز 1000 Ccoin = 10 Ccoin کارمزد).  
  - کارمزد برداشت: 1% (مثلاً برداشت 1000 Ccoin = 10 Ccoin کارمزد).  
  - حداقل واریز/برداشت: 100 Ccoin.  
- **پیام نمونه**:  
  ```
  🏦 بانک تو:
  💰 موجودی: 100 Ccoin
  📥 واریز: 1000 Ccoin واریز کردی (کارمزد: 10 Ccoin)
  📤 برداشت: 500 Ccoin برداشت کردی (کارمزد: 5 Ccoin)
  📅 سود بعدی: 29 Ccoin (29 روز باقی‌مونده)
  ```
- **محدودیت‌ها**:  
  - کپچا برای واریز/برداشت بالای 5000 Ccoin.  
- **نکات توسعه**:  
  - یه فیلد برای بانک و تاریخ آخرین سود بساز:  
    ```javascript
    userSchema.add({
        bank: { type: Number, default: 0 },
        lastProfit: { type: Date, default: Date.now }
    });
    ```
  - برای سود ماهانه، یه تایمر پیاده‌سازی کن:  
    ```javascript
    schedule.scheduleJob('0 0 1 * *', () => {
        users.updateMany({}, { $inc: { bank: bank * 0.02 } });
    });
    ```

#### 3. 💎 کریستال‌ها (Crystals)
- **هدف**: ارز ویژه برای خرید آیتم‌های خاص.  
- **مکانیزم‌ها**:  
  - تبدیل Ccoin به کریستال: 1000 Ccoin = 10 کریستال (کارمزد 5%).  
  - استفاده در فروشگاه برای آیتم‌های ویژه (مثلاً نقش افسانه‌ای).  
- **پیام نمونه**:  
  ```
  💎 کریستال‌ها:
  📥 1000 Ccoin تبدیل کردی (کارمزد: 50 Ccoin)
  📤 10 کریستال دریافت کردی!
  💎 موجودی: 10 کریستال
  ```
- **محدودیت‌ها**:  
  - حداکثر تبدیل روزانه: 5000 Ccoin (50 کریستال).  
- **نکات توسعه**:  
  - یه فیلد برای کریستال‌ها بساز:  
    ```javascript
    userSchema.add({ crystals: { type: Number, default: 0 } });
    ```

#### 4. 💸 انتقال سکه (Transfer)
- **هدف**: انتقال Ccoin بین کاربران.  
- **مکانیزم‌ها**:  
  - کارمزد انتقال: 2% (مثلاً انتقال 1000 Ccoin = 20 Ccoin کارمزد).  
  - کپچا برای انتقال بالای 1000 Ccoin.  
  - ثبت تراکنش تو تاریخچه.  
- **پیام نمونه**:  
  ```
  💸 انتقال سکه:
  📤 1000 Ccoin به @User2 فرستادی (کارمزد: 20 Ccoin)
  💰 موجودی کیف پولت: 480 Ccoin
  ```
- **محدودیت‌ها**:  
  - حداکثر انتقال روزانه: 5000 Ccoin.  
  - حداقل انتقال: 50 Ccoin.  
- **نکات توسعه**:  
  - یه سیستم کپچا پیاده‌سازی کن:  
    ```javascript
    const code = Math.floor(Math.random() * 10000).toString().padStart(4, '0');
    await interaction.reply({ content: `لطفاً کد ${code} رو وارد کن!`, ephemeral: true });
    ```

#### 5. 🎁 پاداش روزانه (Daily Reward)
- **هدف**: تشویق کاربران به فعالیت روزانه با سیستم استریک (Streak).  
- **مکانیزم‌ها**:  
  - جایزه روزانه: 50 Ccoin.  
  - استریک: اگر 7 روز متوالی جایزه بگیری، 200 Ccoin جایزه اضافه می‌گیره.  
  - ریست استریک: اگر یه روز جایزه نگیری، استریک به 0 برمی‌گرده.  
- **پیام نمونه**:  
  ```
  🎁 پاداش روزانه:
  💰 50 Ccoin جایزه گرفتی!
  📅 روز 3 از 7 (4 روز تا جایزه ویژه 200 Ccoin)
  ⏰ پاداش بعدی: 23 ساعت و 59 دقیقه دیگه
  ```
- **محدودیت‌ها**:  
  - هر 24 ساعت یه بار.  
- **نکات توسعه**:  
  - فیلد استریک و زمان آخرین پاداش رو اضافه کن:  
    ```javascript
    userSchema.add({
        dailyStreak: { type: Number, default: 0 },
        lastDaily: { type: Date }
    });
    ```
  - برای ریست استریک، زمان آخرین پاداش رو چک کن:  
    ```javascript
    if (Date.now() - user.lastDaily > 48 * 60 * 60 * 1000) {
        user.dailyStreak = 0;
    }
    ```

#### 6. 📜 سیستم تراکنش (Transactions)
- **هدف**: ثبت تاریخچه تمام تراکنش‌های مالی برای شفافیت.  
- **مکانیزم‌ها**:  
  - ثبت هر تراکنش (واریز، برداشت، انتقال، خرید و...).  
  - نمایش 10 تراکنش آخر کاربر.  
- **پیام نمونه**:  
  ```
  📜 تاریخچه تراکنش‌ها:
  1. 📤 انتقال 1000 Ccoin به @User2 (کارمزد: 20 Ccoin) - 1403/07/01
  2. 📥 واریز 500 Ccoin به بانک (کارمزد: 5 Ccoin) - 1403/07/01
  3. 🎁 پاداش روزانه: 50 Ccoin - 1403/07/01
  ```
- **نکات توسعه**:  
  - یه دیتابیس جدا برای تراکنش‌ها بساز:  
    ```javascript
    const transactionSchema = new mongoose.Schema({
        userId: String,
        type: String, // "transfer", "deposit", "daily", ...
        amount: Number,
        targetId: String, // برای انتقال
        fee: Number,
        timestamp: { type: Date, default: Date.now }
    });
    ```

## 2️⃣ 🎮 بازی‌ها (Games)
### 🎯 هدف
سرگرمی و کسب Ccoin از طریق بازی‌های متنوع و رقابتی.

### 📜 Embed بخش بازی‌ها
- **عنوان**:  
  `🎮 بخش بازی‌ها`
- **توضیحات**:  
  `با بازی‌های متنوع Ccoin جمع کن و با دوستات رقابت کن! 🏆`
- **رنگ Embed**: قرمز (#FF0000) برای حس هیجان.

#### 1. 🎲 پرتاب سکه (Coinflip)
- **هدف**: شرط‌بندی روی شیر یا خط.  
- **مکانیزم‌ها**:  
  - هزینه: 20 Ccoin.  
  - شانس برد: 50% (برد = 40 Ccoin).  
  - انیمیشن ساده: نمایش پرتاب سکه با تأخیر 2 ثانیه.  
- **پیام نمونه**:  
  ```
  🎲 پرتاب سکه:
  🔄 سکه پرتاب شد...
  ✅ شیر! 40 Ccoin برنده شدی!
  💰 موجودی کیف پولت: 520 Ccoin
  ```
- **محدودیت‌ها**:  
  - هر 5 دقیقه یه بار.  
- **نکات توسعه**:  
  - یه سیستم تصادفی ساده پیاده‌سازی کن:  
    ```javascript
    const result = Math.random() < 0.5 ? 'شیر' : 'خط';
    const win = result === userChoice;
    ```

#### 2. ✂️ سنگ کاغذ قیچی (RPS)
- **هدف**: بازی کلاسیک با ربات یا کاربر دیگه.  
- **مکانیزم‌ها**:  
  - هزینه: 20 Ccoin.  
  - انتخاب: سنگ، کاغذ یا قیچی.  
  - بازی با ربات: شانس برد 33% (تساوی 33%).  
  - بازی با کاربر: هزینه 20 Ccoin برای هر نفر، برنده 35 Ccoin (کارمزد 10%).  
- **پیام نمونه**:  
  ```
  ✂️ سنگ کاغذ قیچی:
  🧑 تو: سنگ | 🤖 ربات: کاغذ
  ❌ باختی! کاغذ سنگ رو می‌بره.
  💰 موجودی کیف پولت: 480 Ccoin
  ```
- **محدودیت‌ها**:  
  - هر 5 دقیقه یه بار.  
- **نکات توسعه**:  
  - برای بازی با کاربر، سیستم Matchmaking پیاده‌سازی کن (تو بخش بعدی توضیح داده می‌شه).  
  - یه منطق ساده برای برد/باخت:  
    ```javascript
    const outcomes = {
        'سنگ': { beats: 'قیچی', loses: 'کاغذ' },
        'کاغذ': { beats: 'سنگ', loses: 'قیچی' },
        'قیچی': { beats: 'کاغذ', loses: 'سنگ' }
    };
    ```

#### 3. 🔢 حدس عدد (Number Guess)
- **هدف**: حدس عدد انتخاب‌شده توسط ربات.  
- **مکانیزم‌ها**:  
  - هزینه: 30 Ccoin.  
  - ربات یه عدد بین 1 تا 100 انتخاب می‌کنه.  
  - کاربر 10 شانس برای حدس داره.  
  - راهنمایی: بعد از هر حدس، ربات می‌گه عدد "بزرگ‌تر" یا "کوچیک‌تر"ه.  
  - جایزه: 100 Ccoin برای برد.  
- **پیام نمونه**:  
  ```
  🔢 حدس عدد:
  📊 یه عدد بین 1 تا 100 حدس بزن! (شانس باقی‌مونده: 10)
  🧑 حدس تو: 50
  🤖 عدد بزرگ‌تره! (شانس باقی‌مونده: 9)
  ```
- **محدودیت‌ها**:  
  - هر 10 دقیقه یه بار.  
- **نکات توسعه**:  
  - یه سیستم ذخیره موقت برای عدد و تعداد شانس‌ها بساز:  
    ```javascript
    const games = new Map(); // { userId: { number: Number, guessesLeft: Number } }
    games.set(userId, { number: Math.floor(Math.random() * 100) + 1, guessesLeft: 10 });
    ```

#### 4. 🎲 مسابقه تاس (Dice Duel)
- **هدف**: رقابت دو نفره با تاس.  
- **مکانیزم‌ها**:  
  - هزینه: 50 Ccoin برای هر نفر.  
  - هر بازیکن یه تاس 6-وجهی می‌ریزه (عدد 1-6).  
  - بازیکن با عدد بالاتر برنده می‌شه.  
  - تساوی: تاس دوباره ریخته می‌شه (حداکثر 3 دور).  
  - جایزه: برنده 80 Ccoin (کارمزد 20%).  
- **پیام نمونه**:  
  ```
  🎲 مسابقه تاس:
  @User1: 4 | @User2: 6
  🏆 @User2 برنده شد و 80 Ccoin گرفت!
  ```
- **محدودیت‌ها**:  
  - هر 5 دقیقه یه بار.  
- **نکات توسعه**:  
  - از سیستم Matchmaking استفاده کن (تو بخش بعدی توضیح داده می‌شه).  

#### 5. 🎡 چرخ شانس (Wheel of Fortune)
- **هدف**: چرخاندن چرخ برای دریافت جوایز متنوع.  
- **مکانیزم‌ها**:  
  - هزینه: 50 Ccoin.  
  - جوایز:  
    - 25 Ccoin (40% شانس).  
    - 100 Ccoin (30% شانس).  
    - 500 Ccoin (15% شانس).  
    - 5 کریستال (10% شانس).  
    - آیتم ویژه (5% شانس).  
  - انیمیشن چرخش: 3 ثانیه تأخیر.  
- **پیام نمونه**:  
  ```
  🎡 چرخ شانس:
  ⏳ چرخ داره می‌چرخه...
  🎉 100 Ccoin برنده شدی!
  💰 موجودی کیف پولت: 550 Ccoin
  ```
- **محدودیت‌ها**:  
  - هر 15 دقیقه یه بار.  
- **نکات توسعه**:  
  - یه سیستم وزن‌دهی برای جوایز پیاده‌سازی کن:  
    ```javascript
    const rewards = [
        { reward: '25 Ccoin', weight: 40 },
        { reward: '100 Ccoin', weight: 30 },
        { reward: '500 Ccoin', weight: 15 },
        { reward: '5 کریستال', weight: 10 },
        { reward: 'آیتم ویژه', weight: 5 }
    ];
    const totalWeight = rewards.reduce((sum, r) => sum + r.weight, 0);
    const random = Math.random() * totalWeight;
    ```

#### 6. 🕵️‍♂️ مافیا (Mafia)
- **هدف**: بازی گروهی استراتژیک.  
- **مکانیزم‌ها**:  
  - حداقل 5 نفر، حداکثر 10 نفر.  
  - هزینه: 50 Ccoin برای هر نفر.  
  - نقش‌ها: مافیا، شهروند، دکتر، کارآگاه.  
  - فازها: شب (مافیا حذف می‌کنه) و روز (رأی‌گیری برای حذف).  
  - جایزه: 100 Ccoin برای هر نفر تو تیم برنده.  
- **پیام نمونه**:  
  ```
  🕵️‍♂️ مافیا:
  شب اول: مافیا @User1 رو حذف کرد!
  روز اول: @User2 حذف شد (شهروند بود)
  🏆 شهروندان برنده شدن! هر نفر 100 Ccoin گرفت!
  ```
- **محدودیت‌ها**:  
  - هر 30 دقیقه یه بار برای هر کاربر.  
- **نکات توسعه**:  
  - یه چنل موقت برای بازی بساز:  
    ```javascript
    guild.channels.create({ name: `mafia-${gameId}`, type: 'GUILD_TEXT' });
    ```

#### 7. ⚔️ بازی‌های رقابتی (Competitive Games)
- **هدف**: رقابت دو نفره یا گروهی برای کسب Ccoin.  
- **بازی‌ها**:  
  - **دوئل**: هزینه 50 Ccoin، شانس برد 50%.  
  - **پوکر سریع**: هزینه 100 Ccoin، جایزه 180 Ccoin.  
  - **مسابقه سرعت تایپ**: هزینه 20 Ccoin، جایزه 50 Ccoin.  
  - **دارت رقابتی**: هزینه 30 Ccoin، جایزه 50 Ccoin.  
  - **بمب زمان‌دار**: هزینه 50 Ccoin، جایزه 80 Ccoin.  
  - **پنالتی شانس**: هزینه 30 Ccoin، جایزه 50 Ccoin.  
  - **تیراندازی هدف**: هزینه 40 Ccoin، جایزه 70 Ccoin.  
- **پیام نمونه**:  
  ```
  ⚔️ دوئل:
  @User1 و @User2 شمشیرها رو کشیدن...
  🏆 @User1 برنده شد و 80 Ccoin گرفت!
  ```
- **محدودیت‌ها**:  
  - هر 5 دقیقه یه بار برای هر بازی.  

#### 8. 🔍 سیستم مچ‌میکینگ (Matchmaking)

  - هر 4 ساعت یه بار.  
  - شانس موفقیت: 50%.  
  - مقدار دزدی: حداکثر 100 Ccoin از کیف پول کاربر دیگه.  
  - جریمه شکست: 200 Ccoin.  
- **پیام نمونه**:  
  ```
  🕵️ دزدی:
  🎯 هدف: @User2
  ✅ موفق شدی! 100 Ccoin دزدیدی!
  💰 موجودی کیف پولت: 600 Ccoin
  ```
- **نکات توسعه**:  
  - یه سیستم دزدی بساز:  
    ```javascript
    userSchema.add({ lastRobbery: Date });
    ```

#### 5. 🎁 ارتباط با ربات گیواوی (Giveaway Bridge)
- **مکانیزم‌ها**:  
  - خرید بلیط گیواوی با Ccoin (500 Ccoin).  
  - بلیط‌ها تو ربات دیگه (Moz) قابل‌استفاده هستن (از طریق API).  
- **پیام نمونه**:  
  ```
  🎁 ارتباط با گیواوی:
  🎟️ یه بلیط گیواوی خریدی (هزینه: 500 Ccoin)
  🔗 می‌تونی تو ربات Moz خرجش کنی!
  ```
- **نکات توسعه**:  
  - یه API برای ارتباط با Moz بساز:  
    ```javascript
    app.post('/redeem-ticket', (req, res) => {
        const { userId, ticketCount } = req.body;
        res.json({ success: true, message: 'بلیط خرج شد!' });
    });
    ```

## 7️⃣ 🛠️ پنل مدیریت (Admin Panel)
### 🎯 هدف
مدیریت کامل ربات توسط ادمین‌ها.

### 📜 Embed پنل مدیریت
- **عنوان**:  
  `🛠️ پنل مدیریت`
- **توضیحات**:  
  `ابزارهای پیشرفته برای مدیریت ربات و کاربران! 🔧`
- **رنگ Embed**: خاکستری (#808080) برای حس حرفه‌ای.

#### 1. 💰 مدیریت اقتصاد (Economy Management)
- **مکانیزم‌ها**:  
  - تنظیم نرخ سود بانک، کارمزدها، مالیات.  
  - توزیع Ccoin یا کریستال به همه کاربران.  
- **پیام نمونه**:  
  ```
  💰 مدیریت اقتصاد:
  📊 نرخ سود بانک: 2% (تغییر به 3%)
  💸 توزیع: 100 Ccoin به همه کاربران
  ```

#### 2. 👤 مدیریت کاربران (User Management)
- **مکانیزم‌ها**:  
  - جستجو: بر اساس ID یا نام.  
  - مسدودسازی/بازنشانی: بن کردن یا ریست موجودی کاربر.  
- **پیام نمونه**:  
  ```
  👤 مدیریت کاربران:
  🔍 جستجو: @User1
  📊 موجودی: 500 Ccoin
  🚫 بن: کاربر بن شد!
  ```

#### 3. 🛍️ مدیریت آیتم‌ها (Item Management)
- **مکانیزم‌ها**:  
  - افزودن/ویرایش/حذف آیتم‌ها تو فروشگاه.  
- **پیام نمونه**:  
  ```
  🛍️ مدیریت آیتم‌ها:
  ➕ اضافه کردن: بلیط قرعه‌کشی (500 Ccoin)
  ✏️ ویرایش: کلید جعبه شانس (قیمت: 300 Ccoin)
  🗑️ حذف: نقش ویژه
  ```

#### 4. 📜 تنظیمات لاگ‌ها (Logs Settings)
- **مکانیزم‌ها**:  
  - تنظیم چنل برای لاگ‌های مختلف (تراکنش، بازی، کلن).  
- **پیام نمونه**:  
  ```
  📜 تنظیمات لاگ‌ها:
  💰 چنل تراکنش‌ها: #transaction-logs
  🎮 چنل بازی‌ها: #game-logs
  ```

#### 5. 📊 آمار و اطلاعات (Statistics)
- **مکانیزم‌ها**:  
  - نمایش آمار: تعداد کاربران، مجموع Ccoin، بازی‌های انجام‌شده.  
- **پیام نمونه**:  
  ```
  📊 آمار و اطلاعات:
  👥 تعداد کاربران: 150
  💰 مجموع Ccoin: 1,200,000
  🎮 بازی‌های انجام‌شده: 500
  ```

#### 6. ⚙️ تنظیمات قابلیت‌ها (Feature Settings)
- **مکانیزم‌ها**:  
  - فعال/غیرفعال کردن بخش‌ها (مثلاً دزدی، لاتاری).  
- **پیام نمونه**:  
  ```
  ⚙️ تنظیمات قابلیت‌ها:
  🕵️ دزدی: غیرفعال شد
  🎟️ لاتاری: فعال شد
  ```

#### 7. 💾 پشتیبان‌گیری (Backup)
- **مکانیزم‌ها**:  
  - تهیه و بازیابی پشتیبان از دیتابیس.  
- **پیام نمونه**:  
  ```
  💾 پشتیبان‌گیری:
  📥 تهیه پشتیبان: backup-1403-07-01.json
  📤 بازیابی: دیتابیس بازیابی شد!
  ```
- **نکات توسعه**:  
  - از MongoDB Backup استفاده کن:  
    ```javascript
    const { exec } = require('child_process');
    exec('mongodump --db ccoin --out backup');
    ```
