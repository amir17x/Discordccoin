## ✨ ویژگی‌های **ربات اقتصادی دیسکورد** ✨  

این ربات یک **سیستم اقتصادی کامل و پیشرفته** را در دیسکورد ارائه می‌دهد که کاربران می‌توانند از طریق روش‌های متنوعی سکه کسب کنند 💰، در بازی‌ها شرکت کنند 🎮، سرمایه‌گذاری کنند 📈، تیم بسازند 🤝 و حتی شانس خود را در قرعه‌کشی امتحان کنند 🎟️! همچنین، سیستم دارای **کنترل تورم، تعادل اقتصادی و امنیت بالا** است.  

---

## 🏦 **1. سیستم اقتصادی پایه**  
🔹 **Holi Coin 🪙** – ارز داخلی سرور که کاربران می‌توانند آن را **کسب، خرج و ذخیره** کنند.  
🔹 **کیف پول 👛** – نگهداری سکه‌های کاربران برای استفاده فوری.  
🔹 **حساب بانکی 🏛️** – امنیت بیشتر برای سکه‌ها و امکان واریز و برداشت.  
🔹 **سرمایه‌گذاری 📊** – امکان سپرده‌گذاری سکه با سود ماهانه **۵٪** (قابل تغییر).  
🔹 **مالیات بر دارایی 💸** – کاربرانی که مقدار زیادی سکه در بانک دارند، باید **درصدی مالیات پرداخت کنند** تا از تورم جلوگیری شود.

## 💵 **2. روش‌های کسب درآمد**  
🔸 **🎁 جایزه روزانه** – دریافت مقدار مشخصی سکه **هر ۲۴ ساعت**.  
🔸 **🎡 گردونه شانس** – یک چرخ شانسی با **جوایز تصادفی** (روزانه یک‌بار).  
🔸 **📜 مأموریت‌های روزانه** – انجام وظایف متنوع برای دریافت پاداش.  
🔸 **🕵️ جیب‌زنی** – دزدیدن سکه از دیگر کاربران (با احتمال شکست!).  
🔸 **🎮 برد در بازی‌ها** – کسب سکه از طریق **پیروزی در رقابت‌ها**.  
🔸 **🏆 شرکت در رویدادهای ویژه** – فرصت‌های خاص برای کسب **پاداش‌های کلان**.  
🔸 **💼 شغل‌های درون سرور** – مانند **مأمور بانک یا نگهبان** که حقوق خاص دارند.

## 📜 **3. سیستم مأموریت‌های روزانه**  
✅ **تخصیص خودکار** – هر روز مأموریت‌های جدید دریافت کن!  
✅ **🔄 انواع مأموریت‌ها**:  
  - ارسال پیام 📩  
  - شرکت در بازی‌ها 🎮

  - پیروزی در دوئل ⚔️  
✅ **🏅 پاداش ویژه** – اگر **همه مأموریت‌ها** را انجام دهی، جایزه‌ای **اضافه‌تر** می‌گیری!

## 🎮 **4. سیستم بازی‌ها**

🪙 **شیر یا خط** – شانست رو امتحان کن، **دو برابر سکه ببر یا بباز!**  
✊✋✌ **سنگ، کاغذ، قیچی** – رقابت با دیگران یا **ربات**.  
🔢 **حدس عدد** – یک عدد **بین ۱ تا ۱۰۰** حدس بزن و برنده شو!  
⚔️ **دوئل** – شرط‌بندی و رقابت ۱ به ۱ با **امکان تماشای دیگران!** 👀  
🏆 **شرط‌بندی روی دوئل دیگران** – می‌توانی روی برنده **شرط ببندی!**

## 🤝 **5. سیستم دوستی و تیم‌ها**  
👥 **لیست دوستان** – امکان ارسال و پذیرش درخواست دوستی.  
⚔️ **ایجاد تیم** – تشکیل **کلن و گروه‌های اقتصادی**.  
👑 **مدیریت تیم** – امکان **دعوت اعضا، اخراج، ارتقا رتبه**.

🎉 **پاداش آنی** – هر دستاورد با **یک جایزه ویژه** همراه است.

🎟️ **قرعه‌کشی هفتگی** – **هر هفته یک برنده خوش‌شانس!**  
📈 **شانس بر اساس سرمایه‌گذاری** – هرچه بیشتر سرمایه‌گذاری کنی، **شانس بیشتری برای برنده‌شدن داری!**

یژگی‌های ربات اقتصادی دیسکورد

این ربات یک سیستم اقتصادی پیشرفته برای دیسکورد ارائه می‌دهد که شامل روش‌های متنوع برای کسب درآمد، رقابت، سرمایه‌گذاری و تعامل با سایر کاربران است. این سیستم دارای مکانیزم‌های کنترل تورم، تعادل اقتصادی و امنیت در برابر سوءاستفاده است.

1. سیستم اقتصادی پایه

Holi Coin (سکه اختصاصی): ارز داخلی سرور که کاربران می‌توانند آن را کسب، خرج و ذخیره کنند.

کیف پول: نگهداری سکه‌های قابل دسترس کاربران.

حساب بانکی: امکان واریز و برداشت سکه برای امنیت بیشتر.

سرمایه‌گذاری: امکان سرمایه‌گذاری سکه با نرخ سود ماهانه 5% (قابل تنظیم).

مالیات بر دارایی: کاربران دارای موجودی زیاد در بانک باید درصدی از آن را به‌عنوان مالیات پرداخت کنند تا از تورم جلوگیری شود.

2. روش‌های کسب درآمد

جایزه روزانه: دریافت سکه رایگان روزانه.

گردونه شانس: چرخاندن گردونه برای دریافت جوایز تصادفی (یک‌بار در روز).

مأموریت‌های روزانه: انجام وظایف مختلف برای دریافت پاداش.

جیب‌زنی: دزدی از سکه‌های سایر کاربران در زمان‌های مشخص.

برد در بازی‌ها: کسب سکه از طریق پیروزی در بازی‌های مختلف.

شرکت در رویدادهای ویژه: کسب درآمد از رویدادهای خاص با محدودیت زمانی.

شغل‌های درون سرور: کاربران می‌توانند نقش‌هایی مثل مأمور بانک یا نگهبان داشته باشند که مزایای اقتصادی دارد.

3. سیستم مأموریت‌های روزانه

تخصیص خودکار: هر روز مأموریت‌های جدید به کاربران داده می‌شود.

انواع مأموریت‌ها: ارسال پیام، شرکت در بازی‌ها، واریز به بانک، جیب‌زنی، خرید از فروشگاه و پیروزی در دوئل.

نوار پیشرفت: نمایش گرافیکی میزان پیشرفت مأموریت‌ها.

پاداش ویژه: در صورت تکمیل تمام مأموریت‌ها در یک روز، پاداش اضافه داده می‌شود.

4. سیستم بازی‌ها

تاس: شرط‌بندی روی نتیجه پرتاب تاس.

شیر یا خط: بازی شانسی با شرط‌بندی.

سنگ، کاغذ، قیچی: بازی با ربات یا دوستان.

حدس عدد: حدس زدن عدد انتخابی بین 1 تا 100.

دوئل: رقابت مستقیم با کاربران دیگر با امکان شرط‌بندی.

افزودن بازی‌های مهارتی: مانند بازی‌های حافظه یا اطلاعات عمومی که نیاز به مهارت داشته باشد.

تماشای دوئل: سایر کاربران بتوانند دوئل‌های فعال را ببینند و روی برنده شرط‌بندی کنند.

5. سیستم دوستی و تیم‌ها

ارسال و پذیرش درخواست دوستی: امکان لیست دوستان برای کاربران.

ایجاد تیم: کاربران می‌توانند تیم‌های جدید ایجاد کنند.

مدیریت تیم: امکان دعوت اعضا، پیوستن و خروج از تیم.

چالش تیمی: رقابت بین تیم‌ها با تعیین برنده بر اساس امتیازات ویژه.

6. سیستم دستاوردها و افتخارات

دستاوردهای متنوع: بر اساس فعالیت‌های کاربران.

پاداش خودکار: هنگام دریافت دستاوردها، کاربران پاداش می‌گیرند.

نمایش در پروفایل: امکان نمایش دستاوردهای کاربران در پروفایل آن‌ها.

7. سیستم جکپات و بخت‌آزمایی

جکپات: کاربران می‌توانند سکه‌های خود را در مخزن جکپات قرار دهند و شانس بردن کل مبلغ را داشته باشند.

بخت‌آزمایی هفتگی: قرعه‌کشی بین خریداران بلیط.

شانس متناسب با سرمایه‌گذاری: احتمال بردن بر اساس مقدار سکه‌های گذاشته‌شده در سیستم.

8. سیستم فروشگاه

آیتم‌های متنوع: کاربران می‌توانند محصولات مختلف را خریداری کنند.

قیمت‌گذاری پویا: قیمت‌ها بر اساس عرضه و تقاضا تغییر می‌کنند.

کوله‌پشتی: نگهداری آیتم‌های خریداری‌شده برای استفاده بعدی.

فروش آیتم‌های اختصاصی: کاربران می‌توانند آیتم‌های سفارشی ایجاد کنند و بفروشند.

9. سیستم جیب‌زنی

پنجره زمانی: امکان جیب‌زنی فقط در زمان‌های خاص فعال باشد.

احتمال شکست: درصدی احتمال دارد که دزدی ناموفق باشد و کاربر جریمه شود.

محافظ شخصی: آیتمی در فروشگاه که از کاربران در برابر دزدی محافظت کند.

محدودیت کاربران جدید: حساب‌های جدید تا 5 روز اول قابل دزدی نیستند.

10. سیستم مدیریت لول و XP

کسب XP: با انجام فعالیت‌های مختلف تجربه کسب می‌شود.

افزایش سطح: سطح کاربران با کسب XP افزایش می‌یابد.

نمایش پیشرفت: یک نوار گرافیکی نمایش میزان XP را نشان می‌دهد.

11. امنیت و مقابله با سوءاستفاده

ضد فیک اکانت: بررسی حساب‌های جدید برای جلوگیری از سوءاستفاده.

محدودیت‌های هوشمند: اگر کاربری در مدت کوتاه مقدار زیادی سکه دریافت کند، حسابش برای بررسی مسدود می‌شود.

محدودیت انتقال سکه: سقف انتقال روزانه برای جلوگیری از پول‌شویی داخلی.

حفاظت از حساب بانکی: امنیت بالاتر برای سکه‌های ذخیره‌شده در بانک.

12. سیستم مدیریت ربات

پنل مدیریت: دسترسی ویژه برای مدیران سرور.

مدیریت سکه: افزودن یا تغییر موجودی کاربران.

مدیریت فروشگاه: افزودن یا حذف آیتم‌ها.

بازنشانی داده‌ها: امکان ریست کردن اطلاعات کاربران.

13. بهینه‌سازی و پایداری

مدیریت خطا: سیستم گسترده مدیریت خطا برای جلوگیری از کرش.

راه‌اندازی مجدد خودکار: ربات به‌صورت منظم ریست می‌شود.

مونیتورینگ آپتایم: سیستم HTTP برای نظارت بر وضعیت ربات.

بهینه‌سازی حافظه: پاکسازی خودکار برای عملکرد بهتر.

مدیریت محدودیت نرخ: جلوگیری از اسپم با محدودیت زمانی بین درخواست‌ها.

14. رابط کاربری تعاملی

دکمه‌های تعاملی: برای اجرای راحت‌تر دستورات.

امبدهای اطلاعاتی: نمایش اطلاعات به شکل زیبا و جذاب.

منوهای تو در تو: سازماندهی بهتر دستورات.

انیمیشن‌های متنی: نمایش متحرک برای جذابیت بیشتر.

این سیستم اقتصادی، یک تجربه جامع و سرگرم‌کننده را برای کاربران دیسکورد ارائه می‌دهد و امکان تنظیمات اختصاصی برای مدیریت بهتر را فراهم می‌کند.

import random
from datetime import datetime

THEFT_WINDOW = [(1, 15), (3, 15), (5, 15)]  # ساعت‌های تصادفی برای دزدی
theft_attempts = {}

@bot.command()
async def steal(ctx, member: discord.Member):
    now = datetime.utcnow().hour
    if not any(start <= now < start + duration for start, duration in THEFT_WINDOW):
        await ctx.send("❌ دزدی در این ساعت مجاز نیست!")
        return

    if ctx.author.id in theft_attempts and theft_attempts[ctx.author.id] == datetime.utcnow().date():
        await ctx.send("🚫 شما امروز قبلاً دزدی انجام داده‌اید!")
        return

    if is_new_account(member):
        await ctx.send("🚧 شما نمی‌توانید از کاربران تازه‌وارد دزدی کنید!")
        return
    
    if get_pocket_balance(member.id) == 0:
        await ctx.send("🤷‍♂️ این کاربر پولی در جیبش ندارد!")
        return

    stolen_amount = min(random.randint(10, 100), get_pocket_balance(member.id))
    update_pocket_balance(member.id, -stolen_amount)
    update_pocket_balance(ctx.author.id, stolen_amount)

    theft_attempts[ctx.author.id] = datetime.utcnow().date()
    await ctx.send(f"💰 شما {stolen_amount} سکه از {member.mention} دزدیدید!")

const { EmbedBuilder } = require('discord.js');
const db = require('quick.db');

const transferCooldown = new Map();

module.exports = {
    name: "transfer",
    description: "انتقال سکه به یک کاربر دیگر",
    execute: async (client, message, args) => {
        const sender = message.author;
        const receiver = message.mentions.users.first();
        const amount = parseInt(args[1]);

        if (!receiver || isNaN(amount) || amount <= 0) {
            return message.reply("🚫 لطفاً یک کاربر معتبر و مقدار صحیح وارد کنید.");
        }

        const lastTransfer = transferCooldown.get(sender.id);
        if (lastTransfer && (Date.now() - lastTransfer) < 86400000) {
            return message.reply("⏳ شما فقط هر 24 ساعت یکبار می‌توانید سکه ارسال کنید.");
        }

        const senderBalance = db.get(`coins_${sender.id}`) || 0;
        if (senderBalance < amount) {
            return message.reply("❌ شما موجودی کافی ندارید!");
        }

        // بررسی حساب جدید بودن
        const accountAge = (Date.now() - receiver.createdAt) / (1000 * 60 * 60 * 24);
        if (accountAge < 7) {
            return message.reply("🚧 حساب گیرنده باید حداقل 7 روز عمر داشته باشد.");
        }

        db.subtract(`coins_${sender.id}`, amount);
        db.add(`coins_${receiver.id}`, amount);
        transferCooldown.set(sender.id, Date.now());

        const embed = new EmbedBuilder()
            .setColor("Green")
            .setTitle("💰 انتقال موفق")
            .setDescription(`${amount} سکه به ${receiver} ارسال شد!`);

        message.channel.send({ embeds: [embed] });
    }
};

module.exports = {
    name: "lootbox",
    description: "باز کردن لوت‌باکس با 100 سکه",
    execute: async (client, message) => {
        const user = message.author;
        const balance = db.get(`coins_${user.id}`) || 0;

        if (balance < 100) {
            return message.reply("❌ شما به 100 سکه برای باز کردن لوت‌باکس نیاز دارید!");
        }

        db.subtract(`coins_${user.id}`, 100);

        const prizes = [
            { amount: 500, chance: 10 },
            { amount: 200, chance: 50 },
            { amount: 100, chance: 20 },
            { amount: 0, chance: 10 }
        ];

        let roll = Math.random() * 100;
        let cumulative = 0;
        let reward = 0;

        for (const prize of prizes) {
            cumulative += prize.chance;
            if (roll <= cumulative) {
                reward = prize.amount;
                break;
            }
        }

        db.add(`coins_${user.id}`, reward);

        message.reply(reward > 0 ? `🎉 شما ${reward} سکه برنده شدید!` : "💀 متاسفانه چیزی برنده نشدید!");
    }

const theftAttempts = new Map();
const theftWindows = [
    { start: 1, duration: 15 }, 
    { start: 3, duration: 15 }, 
    { start: 5, duration: 15 }
];

module.exports = {
    name: "steal",
    description: "دزدی از کاربران!",
    execute: async (client, message) => {
        const user = message.author;
        const target = message.mentions.users.first();

        if (!target) {
            return message.reply("🚨 لطفاً یک کاربر را برای دزدی انتخاب کنید.");
        }

        const nowHour = new Date().getHours();
        const inTheftWindow = theftWindows.some(w => nowHour >= w.start && nowHour < w.start + w.duration);

        if (!inTheftWindow) {
            return message.reply("❌ دزدی در این ساعت مجاز نیست!");
        }

        if (theftAttempts.get(user.id) === new Date().toDateString()) {
            return message.reply("🚫 شما امروز قبلاً دزدی کرده‌اید!");
        }

        const accountAge = (Date.now() - target.createdAt) / (1000 * 60 * 60 * 24);
        if (accountAge < 5) {
            return message.reply("🚧 نمی‌توانید از کاربران تازه‌وارد دزدی کنید!");
        }

        const targetBalance = db.get(`pocket_${target.id}`) || 0;
        if (targetBalance === 0) {
            return message.reply("🤷‍♂️ این کاربر پولی در جیبش ندارد!");
        }

        const stolenAmount = Math.min(Math.floor(Math.random() * 100) + 10, targetBalance);
        db.subtract(`pocket_${target.id}`, stolenAmount);
        db.add(`pocket_${user.id}`, stolenAmount);

        theftAttempts.set(user.id, new Date().toDateString());

        message.reply(`💰 شما ${stolenAmount} سکه از ${target} دزدیدید!`);
    }
};

    description: "شرط‌بندی روی شیر یا خط",

        const bet = parseInt(args[0]);

        if (!bet || bet <= 0) return message.reply("🚨 مقدار معتبر وارد کنید!");

        else db.subtract(`coins_${user.id}`, bet);

        message.reply(`🎲 نتیجه: **${flip}**\n💰 موجودی جدید: ${db.get(`coins_${user.id}`)}`);

module.exports = {
    name: "rps",
    description: "بازی سنگ، کاغذ، قیچی",
    execute: async (client, message, args) => {
        const choices = ["سنگ", "کاغذ", "قیچی"];
        const userChoice = args[0];

        if (!choices.includes(userChoice)) {
            return message.reply("🛑 لطفاً یکی از گزینه‌های **سنگ**، **کاغذ** یا **قیچی** را وارد کنید.");
        }

        const botChoice = choices[Math.floor(Math.random() * choices.length)];
        let result = "✋ مساوی شد!";

        if (
            (userChoice === "سنگ" && botChoice === "قیچی") ||
            (userChoice === "کاغذ" && botChoice === "سنگ") ||
            (userChoice === "قیچی" && botChoice === "کاغذ")
        ) {
            result = "🎉 شما بردید!";
            db.add(`coins_${message.author.id}`, 50);
        } else if (userChoice !== botChoice) {
            result = "😢 شما باختید!";
            db.subtract(`coins_${message.author.id}`, 50);
        }

        message.reply(`🤖 من انتخاب کردم: **${botChoice}**\n🏆 نتیجه: ${result}`);
    }
};

ویژگی‌های حفظ کاربران و انگیزه‌دهی: می‌تونی پاداش‌هایی برای فعالیت مداوم کاربران طراحی کنی. مثل رتبه‌بندی‌هایی که کاربران رو براساس میزان فعالیت‌شون در بازی یا در چت سرور رتبه‌بندی می‌کنه.

سیستم بدهی و قرض: می‌تونی اجازه بدی که کاربران از ارز داخلی قرض بگیرن و در صورت عدم پرداخت در زمان مشخص، جریمه بشن یا حتی از امکانات خاص محروم بشن.

### 🏦 **1. سیستم وام و قرض‌گیری (Loan System)**  
کاربران می‌توانند از بانک وام دریافت کنند، اما باید با سود بازپرداخت کنند!  

**📌 ویژگی‌ها:**  
- **انواع وام:**  
  - 🟢 وام کم‌ریسک (سود ۵٪ ماهانه، حداکثر ۱۰۰۰ سکه)  
  - 🟡 وام متوسط (سود ۱۰٪ ماهانه، حداکثر ۵۰۰۰ سکه)  
  - 🔴 وام پرریسک (سود ۲۰٪ ماهانه، حداکثر ۱۰۰۰۰ سکه)  
- **جریمه تأخیر:** اگر کاربر تا موعد مقرر وام را بازپرداخت نکند، **۱۰٪ از موجودی کیف پولش به صورت خودکار کسر می‌شود**.  
- **محدودیت:** هر کاربر همزمان فقط می‌تواند یک وام فعال داشته باشد.  

**💻 کد نمونه (وام کم‌ریسک):**  
```javascript
const { SlashCommandBuilder } = require('discord.js');

module.exports = {
  data: new SlashCommandBuilder()
    .setName('loan')
    .setDescription('دریافت وام از بانک')
    .addStringOption(option =>
      option.setName('type')
        .setDescription('نوع وام')
        .setRequired(true)
        .addChoices(
          { name: 'کم‌ریسک', value: 'low' },
          { name: 'متوسط', value: 'medium' },
          { name: 'پرریسک', value: 'high' }
        ))
    .addIntegerOption(option =>
      option.setName('amount')
        .setDescription('مبلغ وام')
        .setRequired(true)),
  async execute(interaction) {
    const type = interaction.options.getString('type');
    const amount = interaction.options.getInteger('amount');
    const userId = interaction.user.id;

    // بررسی محدودیت‌های وام
    if (type === 'low' && amount > 1000) {
      return interaction.reply('❌ حداکثر وام کم‌ریسک ۱۰۰۰ سکه است!');
    }

    // واریز وام به حساب کاربر
    await updateUserBalance(userId, amount);
    await addLoan(userId, amount, type);

    interaction.reply(`✅ وام ${amount} سکه با موفقیت دریافت شد! (نوع: ${type})`);
  }
};
```

```javascript
    const reason = interaction.options.getString('reason');
    await jailUser(target.id, reason);

    interaction.reply(`✅ ${target.username} به دلیل **${reason}** زندانی شد!`);
  }
};
```

### 📊 **7. سیستم رتبه‌بندی و لیگ‌ها (Ranking System)**  
رقابت برای قرارگیری در جدول برترین‌ها!  

**📌 لیگ‌ها:**  
- 🥉 لیگ برنز (سطح ۱-۱۰)  
- 🥈 لیگ نقره (سطح ۱۱-۲۰)  
- 🥇 لیگ طلا (سطح ۲۱-۳۰)  
- 💎 لیگ الماس (سطح ۳۱+)  

**پاداش‌های لیگ:**  
- 💰 سکه رایگان هر هفته بر اساس رتبه.  
- 🛡️ آیتم‌های انحصاری برای لیگ الماس.  
- 🏆 نشان ویژه در پروفایل.  

**💻 کد نمونه (بررسی رتبه):**  
```javascript
function getUserLeague(level) {
  if (level <= 10) return 'برنز';
  else if (level <= 20) return 'نقره';
  else if (level <= 30) return 'طلا';
  else return 'الماس';
}

// نمایش رتبه در پروفایل
client.on('interactionCreate', async (interaction) => {
  if (interaction.commandName === 'profile') {
    const user = interaction.user;
    const level = getUserLevel(user.id);
    const league = getUserLeague(level);

    interaction.reply({
      embeds: [{
        title: `${user.username} - سطح ${level}`,
        description: `🏆 لیگ فعلی: **${league}**`
      }]
    });
  }
});
```
