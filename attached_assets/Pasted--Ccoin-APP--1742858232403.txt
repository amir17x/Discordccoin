### 🎮 مستندات جامع بازی‌های رقابتی (Competitive Games) ربات Ccoin APP
#### 🌟 هدف بازی‌های رقابتی
بازی‌های رقابتی برای ایجاد رقابت هیجان‌انگیز بین کاربران طراحی شدن و هدفشون اینه که کاربران بتونن با مهارت، شانس یا استراتژی خودشون Ccoin و کریستال به دست بیارن، رتبه‌بندی بشن و حس رقابت و سرگرمی رو تجربه کنن. این بازی‌ها برای 2 نفر یا بیشتر طراحی شدن و شامل مکانیزم **جستجوی رقیب** (Matchmaking) هستن که کاربر می‌تونه یا یه رقیب رو دعوت کنه یا به‌صورت تصادفی با یه نفر مچ بشه. 🏆

#### 🌟 منوی جستجوی رقیب (Matchmaking)
- **پیام**:  
  `"🔍 یه رقیب برای بازی انتخاب کن! 🎮\nمی‌تونی یه نفر رو دعوت کنی یا به‌صورت تصادفی مچ بشی!"`  
- **دکمه‌ها**:  
  1. 🔎 **جستجوی تصادفی** (سبز 🟩)  
     - کاربر تو صف انتظار قرار می‌گیره و با یه نفر که تو همون بازی درحال جستجوست مچ می‌شه.  
  2. 📩 **دعوت رقیب** (آبی 🟦)  
     - کاربر می‌تونه یه نفر رو از لیست آنلاین‌ها انتخاب کنه و دعوت بفرسته.  
  3. 🔙 **بازگشت** (خاکستری ⬜)  
- **ویژگی‌ها**:  
  - اگر کاربر "جستجوی تصادفی" رو انتخاب کنه، تو یه صف انتظار (Queue) قرار می‌گیره.  
  - اگر 30 ثانیه رقیب پیدا نشه، پیام می‌گیره: `"⏰ رقیب پیدا نشد! دوباره امتحان کن یا یه نفر رو دعوت کن."`  
  - اگر "دعوت رقیب" رو انتخاب کنه، یه لیست از کاربران آنلاین نمایش داده می‌شه (مثلاً با Select Menu).  
  - وقتی رقیب پیدا شد، بازی شروع می‌شه و نتیجه به‌صورت خصوصی برای هر دو بازیکن نمایش داده می‌شه.

### 🛠️ مکانیزم جستجوی رقیب (Matchmaking System)
#### 🌟 جستجوی تصادفی (Random Matchmaking)
- **نحوه کار**:  
  - کاربر روی "جستجوی تصادفی" کلیک می‌کنه و تو صف انتظار برای اون بازی خاص (مثلاً تاس دو نفره) قرار می‌گیره.  
  - سیستم هر 5 ثانیه چک می‌کنه که آیا کاربر دیگه‌ای تو صف هست یا نه.  
  - اگر دو نفر تو صف باشن، مچ می‌شن و بازی شروع می‌شه.  
  - اگر بعد از 30 ثانیه رقیب پیدا نشه، کاربر از صف خارج می‌شه و می‌تونه دوباره امتحان کنه.  
- **پیام درحال جستجو**:  
  `"🔍 درحال جستجوی رقیب... لطفاً صبر کن! (حداکثر 30 ثانیه)"`  
- **پیام مچ شدن**:  
  `"🎉 رقیبت پیدا شد! @User2 آماده‌ست! بازی شروع می‌شه... 🚀"`  

#### 🌟 دعوت رقیب (Invite Opponent)
- **نحوه کار**:  
  - کاربر روی "دعوت رقیب" کلیک می‌کنه.  
  - یه Select Menu با لیست کاربران آنلاین (مثلاً حداکثر 25 نفر) نمایش داده می‌شه.  
  - کاربر یه نفر رو انتخاب می‌کنه و دعوت می‌فرسته.  
  - کاربر دعوت‌شده یه پیام خصوصی می‌گیره:  
    `"📩 @User1 تو رو به یه بازی [تاس دو نفره] دعوت کرده! قبول می‌کنی؟ ✅ رد می‌کنی؟ ❌"`  
  - اگر کاربر دعوت رو قبول کنه، بازی شروع می‌شه.  
  - اگر رد کنه یا 30 ثانیه جوابی نده، دعوت منقضی می‌شه و پیام می‌گیره:  
    `"⏰ دعوت منقضی شد! یه نفر دیگه رو دعوت کن."`  
- **پیام دعوت موفق**:  
  `"🎉 @User2 دعوتت رو قبول کرد! بازی شروع می‌شه... 🚀"`

### 1️⃣ 🎲 تاس دو نفره
**هدف**: یه بازی ساده و سریع برای رقابت بین دو نفر با تاس.  
- **نحوه بازی**:  
  - هر بازیکن 50 Ccoin ورودی می‌ده.  
  - هر نفر یه تاس 6-وجهی می‌ریزه (عدد بین 1 تا 6).  
  - بازیکنی که عدد بالاتری بیاره برنده می‌شه.  
  - در صورت تساوی، تاس دوباره ریخته می‌شه (حداکثر 3 دور تساوی).  
- **جوایز**:  
  - برنده: 80 Ccoin (80% از مجموع ورودی‌ها).  
  - بازنده: 0 Ccoin.  
  - 20% کارمزد (20 Ccoin) به سیستم اقتصادی ربات می‌ره.  
- **پیام نمونه**:  
  `"🎲 @User1: 4 | @User2: 6\n@User2 برنده شد و 80 Ccoin گرفت! 🏆"`  
- **ویژگی‌ها**:  
  - **شانس تساوی**: 16.67% (1/6).  
  - **مدت زمان**: حداکثر 30 ثانیه برای هر دور (شامل جستجوی رقیب).  
  - **محدودیت**: هر کاربر می‌تونه هر 5 دقیقه یه بار این بازی رو انجام بده.  
- **لاگ**:  
  - ثبت نتیجه: `"@User1 و @User2 تاس دو نفره بازی کردن | برنده: @User2 | جایزه: 80 Ccoin"`

### 6️⃣ 🕵️‍♂️ مافیا
**هدف**: یه بازی گروهی و استراتژیک برای رقابت و تعامل (بازی گروهی که تو منوی رقابتی هم قرار داره).  
- **نحوه بازی**:  
  - حداقل 5 نفر، حداکثر 10 نفر (برای مافیا تو بخش رقابتی، می‌تونه به‌صورت 2 نفره هم طراحی بشه).  
  - هر بازیکن 50 Ccoin ورودی می‌ده.  
  - نقش‌ها به‌صورت تصادفی分配 می‌شن:  
    - **شهروند**: باید مافیا رو پیدا کنه.  
    - **مافیا**: هر شب یه نفر رو حذف می‌کنه (1 مافیا برای 2 نفر).  
    - **دکتر**: هر شب می‌تونه یه نفر رو نجات بده (فقط برای 5 نفر یا بیشتر).  
    - **کارآگاه**: هر شب می‌تونه نقش یه نفر رو ببینه (فقط برای 5 نفر یا بیشتر).  
  - برای 2 نفر: فقط مافیا و شهروند (بدون نقش‌های اضافی).  
  - بازی در دو فاز انجام می‌شه:  
    - **شب**: مافیا تصمیم می‌گیره کی رو حذف کنه (در دایرکت).  
    - **روز**: بازیکن‌ها بحث می‌کنن و با رأی‌گیری یه نفر رو حذف می‌کنن.  
  - بازی تا وقتی ادامه داره که یا مافیا برنده بشه (تعداد مافیا = تعداد شهروندان) یا شهروندان برنده بشن (همه مافیا حذف بشن).  
- **جستجوی رقیب برای 2 نفر**:  
  - برای حالت 2 نفره، جستجوی رقیب مثل بقیه بازی‌ها عمل می‌کنه (جستجوی تصادفی یا دعوت).  
  - برای حالت گروهی (5 نفر یا بیشتر)، یه چنل موقت برای بازی ایجاد می‌شه و کاربران تو صف انتظار قرار می‌گیرن.  
- **جوایز**:  
  - تیم برنده (مافیا یا شهروندان): هر نفر 100 Ccoin.  
  - تیم بازنده: 0 Ccoin.  
  - 20% از مجموع ورودی‌ها (مثلاً 20 Ccoin برای 2 نفر) به سیستم اقتصادی ربات می‌ره.  
- **پیام نمونه (2 نفر)**:

- **پیام نمونه (گروهی)**:  
  `"🕵️‍♂️ شب اول: مافیا @User1 رو حذف کرد!\nروز اول: همه رأی دادن و @User2 حذف شد (شهروند بود)\nشهروندان برنده شدن! هر نفر 100 Ccoin گرفت! 🏆"`

- **پیام نمونه**:

### 🛠️ نکات اضافی برای توسعه‌دهنده
#### 🌟 پیاده‌سازی سیستم Matchmaking
برای پیاده‌سازی سیستم جستجوی رقیب، می‌تونی از یه ساختار داده مثل **Array** یا **Map** برای مدیریت صف انتظار استفاده کنی. یه نمونه کد ساده با Discord.js:

```javascript
const { Client, ButtonBuilder, ButtonStyle, ActionRowBuilder, EmbedBuilder } = require('discord.js');
const client = new Client({ intents: ['Guilds', 'GuildMessages', 'MessageContent'] });

// صف انتظار برای هر بازی
const queues = new Map(); // { gameName: [{ userId, interaction }, ...] }

client.on('interactionCreate', async (interaction) => {
    if (!interaction.isButton()) return;

    // وقتی کاربر روی یه بازی (مثلاً تاس دو نفره) کلیک می‌کنه
    if (interaction.customId === 'dice_game') {
        const randomButton = new ButtonBuilder()
            .setCustomId('random_match_dice')
            .setLabel('جستجوی تصادفی')
            .setStyle(ButtonStyle.Success)
            .setEmoji('🔎');

        const inviteButton = new ButtonBuilder()
            .setCustomId('invite_opponent_dice')
            .setLabel('دعوت رقیب')
            .setStyle(ButtonStyle.Primary)
            .setEmoji('📩');

        const backButton = new ButtonBuilder()
            .setCustomId('back_to_competitive')
            .setLabel('بازگشت')
            .setStyle(ButtonStyle.Secondary)
            .setEmoji('🔙');

        const row = new ActionRowBuilder().addComponents(randomButton, inviteButton, backButton);

        await interaction.update({
            embeds: [new EmbedBuilder()
                .setTitle('🔍 یه رقیب برای تاس دو نفره انتخاب کن!')
                .setDescription('می‌تونی یه نفر رو دعوت کنی یا به‌صورت تصادفی مچ بشی!')
                .setColor(0x00FF00) // سبز
            ],
            components: [row],
            ephemeral: true
        });
    }

    // جستجوی تصادفی
    if (interaction.customId === 'random_match_dice') {
        const gameName = 'dice_game';
        if (!queues.has(gameName)) queues.set(gameName, []);

        const queue = queues.get(gameName);
        queue.push({ userId: interaction.user.id, interaction });

        if (queue.length >= 2) {
            // دو نفر پیدا شدن، مچ کردن
            const player1 = queue.shift();
            const player2 = queue.shift();

            // شروع بازی (مثلاً تاس دو نفره)
            const dice1 = Math.floor(Math.random() * 6) + 1;
            const dice2 = Math.floor(Math.random() * 6) + 1;
            const winner = dice1 > dice2 ? player1 : dice2 > dice1 ? player2 : null;

            const resultMessage = winner
                ? `🎲 <@${player1.userId}>: ${dice1} | <@${player2.userId}>: ${dice2}\n<@${winner.userId}> برنده شد و 80 Ccoin گرفت! 🏆`
                : `🎲 <@${player1.userId}>: ${dice1} | <@${player2.userId}>: ${dice2}\nتساوی شد! ورودی‌ها برگشت داده شد.`;

            // ارسال نتیجه به هر دو بازیکن
            await player1.interaction.update({
                embeds: [new EmbedBuilder().setDescription(resultMessage).setColor(0xFFFF00)],
                components: [],
                ephemeral: true
            });
            await player2.interaction.update({
                embeds: [new EmbedBuilder().setDescription(resultMessage).setColor(0xFFFF00)],
                components: [],
                ephemeral: true
            });
        } else {
            // درحال جستجو
            await interaction.update({
                embeds: [new EmbedBuilder()
                    .setDescription('🔍 درحال جستجوی رقیب... لطفاً صبر کن! (حداکثر 30 ثانیه)')
                    .setColor(0xFFFF00)
                ],
                components: [],
                ephemeral: true
            });

            // بعد از 30 ثانیه اگر رقیب پیدا نشد
            setTimeout(async () => {
                const queue = queues.get(gameName);
                const userIndex = queue.findIndex(q => q.userId === interaction.user.id);
                if (userIndex !== -1) {
                    queue.splice(userIndex, 1);
                    await interaction.editReply({
                        embeds: [new EmbedBuilder()
                            .setDescription('⏰ رقیب پیدا نشد! دوباره امتحان کن یا یه نفر رو دعوت کن.')
                            .setColor(0xFF0000)
                        ],
                        components: [],
                        ephemeral: true
                    });
                }
            }, 30000);
        }
    }

    // دعوت رقیب (به زودی اضافه می‌شه)
    if (interaction.customId === 'invite_opponent_dice') {
        // اینجا می‌تونی یه Select Menu برای انتخاب کاربر آنلاین اضافه کنی
        await interaction.update({
            embeds: [new EmbedBuilder()
                .setDescription('📩 یه نفر رو از لیست انتخاب کن و دعوت بفرست!')
                .setColor(0x0000FF)
            ],
            components: [], // Select Menu برای انتخاب کاربر
            ephemeral: true
        });
    }
});
```
