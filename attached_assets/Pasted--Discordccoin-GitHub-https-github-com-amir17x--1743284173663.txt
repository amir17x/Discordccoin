### توضیح جامع برای مستندسازی README

#### ۱. عنوان و توضیح کلی پروژه
- **عنوان:** یه عنوان جذاب و مرتبط با پروژه، مثلاً "Discordccoin: ربات دیسکورد چندمنظوره برای اخبار، ارز و سرگرمی".
- **توضیح کلی:** توضیح بده که این پروژه چیه، چه هدفی داره و چه مشکلاتی رو حل می‌کنه. مثلاً:  
  "Discordccoin یه ربات دیسکورد قدرتمنده که با استفاده از APIهای رایگان، قابلیت‌هایی مثل نمایش اخبار (مثلاً از بی‌بی‌سی فارسی)، قیمت ارزهای دیجیتال، اطلاعات بازی‌ها و سرگرمی‌های متنوع رو به کاربرا ارائه می‌ده. این ربات برای سرورهای دیسکورد طراحی شده تا تجربه‌ای جذاب و کاربردی برای کاربرا فراهم کنه."

#### ۲. ویژگی‌ها (Features)
لیست ویژگی‌های فعلی و برنامه‌ریزی‌شده‌ی پروژه رو بنویس:
- نمایش اخبار به‌روز از منابع معتبر (مثل بی‌بی‌سی فارسی با NewsAPI).
- نمایش قیمت و اطلاعات ارزهای دیجیتال (مثل بیت‌کوین با CoinGecko API).
- اطلاعات بازی‌ها و معرفی بازی‌های رایگان (با RAWG API و FreeToGame API).
- سرگرمی‌هایی مثل جوک، میم و تصاویر تصادفی (با JokeAPI، Meme API و TheCatAPI).
- قابلیت‌های آینده: اضافه کردن اطلاعات آب‌وهوا، نقل‌قول‌های الهام‌بخش، و دستور پخت غذا.

#### ۳. پیش‌نیازها (Prerequisites)
کاربرا و توسعه‌دهنده‌ها باید چه چیزهایی رو آماده کنن تا بتونن پروژه رو اجرا کنن:
- Node.js (نسخه 16 یا بالاتر).
- npm یا yarn برای مدیریت پکیج‌ها.
- یه حساب دیسکورد و توکن ربات (Bot Token) از Discord Developer Portal.
- API Key برای APIهایی که استفاده می‌کنی (مثل NewsAPI، CoinGecko، و غیره).
- (اختیاری) VPN برای دسترسی به APIهایی که ممکنه برای IPهای ایرانی محدود باشن.

#### ۴. نصب و راه‌اندازی (Installation)
یه راهنمای قدم‌به‌قدم برای نصب و اجرای پروژه:
1. ریپازیتوری رو کلون کن:  
   ```
   git clone https://github.com/amir17x/Discordccoin.git
   ```
2. به دایرکتوری پروژه برو:  
   ```
   cd Discordccoin
   ```
3. پکیج‌ها رو نصب کن:  
   ```
   npm install
   ```
4. فایل محیط (`.env`) رو بساز و متغیرهای لازم رو تنظیم کن:  
   ```
   DISCORD_TOKEN=your_discord_bot_token
   NEWSAPI_KEY=your_newsapi_key
   COINGECKO_API_KEY=your_coingecko_key
   ```
5. پروژه رو اجرا کن:  
   ```
   npm run dev
   ```
6. ربات رو به سرور دیسکوردت دعوت کن و ازش استفاده کن!

#### ۵. ساختار پروژه (Project Structure)
ساختار دایرکتوری‌ها و فایل‌ها رو توضیح بده (حتی اگه هنوز کامل پیاده‌سازی نشده باشن):
- **`src/`**: شامل کدهای اصلی ربات.
  - `src/index.ts`: فایل اصلی که ربات رو راه‌اندازی می‌کنه.
  - `src/commands/`: دایرکتوری دستورات ربات (مثل `news.ts`، `price.ts`، `joke.ts`).
  - `src/api/`: دایرکتوری برای مدیریت درخواست‌های API (مثل `newsApi.ts`، `coinGeckoApi.ts`).
  - `src/utils/`: ابزارهای کمکی (مثل مدیریت کش، هندل کردن خطاها).
- **`vite.config.ts`**: فایل تنظیمات Vite برای باندل کردن پروژه (در حال حاضر موجوده).
- **`.env`**: فایل محیط برای ذخیره متغیرهای حساس مثل توکن‌ها و API Keyها.
- **`package.json`**: شامل وابستگی‌ها و اسکریپت‌های پروژه.
